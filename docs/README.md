# AI Agents Blog - Static Build in /docs

This directory contains the complete static build of the AI Agents Blog, automatically generated by GitHub Actions.

## 🚀 Automatic Deployment

The blog is automatically built and deployed via GitHub Actions on every push to the `main` branch:

1. **Push code** → GitHub Actions triggers
2. **Build Next.js** → Static site generated
3. **Output to /docs** → All files placed here
4. **Deploy to GitHub Pages** → Site goes live

## 📁 Structure

- `index.html` - Beautiful homepage with all 9 AI agent posts
- `styles.css` - CSS styles for the static site
- `about.html` - About page
- `posts/` - Individual post pages (generated by Next.js)
- This README

## 🔧 How It Works

### Automatic Posts Index
- All `.md` files in `/posts` are automatically indexed
- `posts.md` is generated in the root with all posts listed
- GitHub Actions updates the index on every build

### Continuous Updates
1. Add new `.md` file to `/posts`
2. Push to GitHub
3. GitHub Actions rebuilds the site
4. New post appears automatically

### Static Site Benefits
- Fast loading (no server-side rendering)
- Can be served from any static host
- GitHub Pages compatible
- SEO friendly

## ⚙️ Configuration

GitHub Pages should be configured to serve from `/docs` directory:

1. Go to repository Settings → Pages
2. Source: Deploy from a branch
3. Branch: `main` → `/docs` folder
4. Save

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Generate posts index
npm run generate:posts-index

# Build to /docs
npm run build:docs

# Or full build with index generation
npm run build:full
```
